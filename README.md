# 📊 Sales Forecasting Dashboard

## 🔍 Project Overview

This project focuses on analyzing historical sales data and building an interactive dashboard to visualize trends and insights. It also includes a simple forecasting model to predict future sales.

The dashboard is developed using Streamlit, allowing users to explore data in an interactive and user-friendly way.

---

## 🚀 Features

* 📁 Upload and view sales dataset
* 📊 Sales trend visualization
* 📅 Monthly sales analysis
* 📈 Key performance metrics (Total, Average, Maximum Sales)
* 🔮 Future sales prediction
* 🎛️ Interactive dashboard interface

---

## 🛠️ Tools and Technologies

* Python
* Pandas
* Streamlit
* Matplotlib
* Scikit-learn
